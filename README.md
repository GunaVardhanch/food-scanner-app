# Food Scanner App

Real-time ingredients analysis and health scoring for Indian food products.

- **Frontend**: Next.js (Static Export)
- **Backend**: FastAPI
- **OCR**: EasyOCR + YOLOv8
- **Intelligence**: Custom NER + Additives Expert System

## How to use
Upload a photo of the ingredients list on any food package to get a detailed health report, risk tier, and AI-driven explanations.
